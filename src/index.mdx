# What is R.A.D. Routes?

---

R.A.D. Routes is a code sharing and execution platform:

- **R.** Random

- **A.** Access

- **D.** Data

- **Routes** The way R.A.D. is accessed via your/the communities routes. Via the R.A.D. Routes Runner, which contacts our RESTful style API to pull your created route from the R.A.D. cloud, and run them on your machine.


## Use Case

---

R.A.D. Routes is ment to be used mostly for shortish, and normally one file scripts that you want to execute frequently on your computer, server, or team projects.


## Current Supported Languages

---

A R.A.D. Route can currently be written, saved, and executed in 
- Python
- Node.js
- Ruby

We are hard at work adding more features and functionality; so please let use know what other languages you would be interested in running R.A.D. Routes with.
